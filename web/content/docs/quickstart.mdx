# Quickstart

## Prerequisites

- macOS on Apple Silicon
- Access to your local development folders
- Claude Code or other local agent tooling (optional)

## Install

```bash
open https://github.com/webrenew/agent-space/releases/latest
```

Download the latest `.dmg`, drag **Agent Space** into `Applications`, and open the app.

## First Run

1. Pick your default workspace directory.
2. Open a project from Explorer.
3. Start a chat scoped to that directory.
4. Confirm activity and token telemetry in the sidebar panels.

## Verify

- Search tab can find files in your project root.
- Office panel shows one active agent for one active chat.
- Token tracker increments for chat/tool actions.
